Django连接MySQL

发表于 2016-12-13 | 修改于 2016-12-13   |   分类于 Python

下载My-python，进入，执行：

python setup.py install

提示，mysql_config not found，应该是mysql_config路径没有设置对，打开~/.bash_profile加上路径：

export PATH=${PATH}:/usr/local/mysql/bin

然后，以管理员身份运行：

sudo python setup.py install

安装成功！

下面设置django的。

使用创建用户，数据库

进入mysql/bin 然后登陆

sudo ./mysql -u root -p

输入密码之后，创建用户

CREATE USER 'username'@'host' IDENTIFIED BY 'password';

创建数据库

CREATE database <database>;

查看所有用户：

select * from mysql.user;

查看所有数据库：

show databases

设置用户权限：

grant all on foo_db.* to 'foo_user'@'主机名';

flush privileges;
