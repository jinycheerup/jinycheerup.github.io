Django连接MySQL

发表于 2016-12-13 | 修改于 2016-12-13   |   分类于 Python

下载My-python，进入，执行：

python setup.py install

提示，mysql_config not found，应该是mysql_config路径没有设置对，打开~/.bash_profile加上路径：

export PATH=${PATH}:/usr/local/mysql/bin

然后，以管理员身份运行：

sudo python setup.py install

安装成功！
